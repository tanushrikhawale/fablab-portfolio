Web Development

Web development is the foundation of modern digital experiences. This 7-day FabLab module guided me through HTML, CSS, and JavaScript. What follows is not just notes—it’s my journey of concepts, struggles, and breakthroughs.

Day 1 – HTML Structure

HTML (HyperText Markup Language) provides the skeleton of every webpage. On Day 1, I explored the anatomy of an HTML document: <!DOCTYPE>, <html>, <head>, and <body>. I learned how browsers interpret markup step by step.

In the FabLab, we built our first page with just a heading and a paragraph. The thrill of seeing "Hello World" appear made me realize how a few lines of text can become a webpage. Initially, I mixed up what goes inside <head> vs <body>, but I understood quickly after debugging. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Day 2 – Common Tags

We dived into headings, paragraphs, lists, links, and images. I discovered how these tags shape content meaningfully. I also practiced creating ordered and unordered lists.

My experience: I was fascinated by how an <a> tag can link pages together, making the web truly interconnected. I accidentally broke an image path, but fixing it taught me about relative vs absolute file references.

Day 3 – Semantic HTML

Semantic tags like <header>, <main>, and <footer> bring structure and accessibility to HTML. Instead of stacking <div>s, semantic elements communicate meaning.

In the session, we built a mini webpage layout. At first, I felt lost with so many new tags, but once arranged, the structure looked natural. Creating my first form using <input> and <button> made me appreciate how user interaction is captured. 

<header>My Website</header>
<main>
  <form>
    <label>Name:</label>
    <input type="text" name="name">
    <button type="submit">Submit</button>
  </form>
</main>
<footer>© 2025</footer>

Day 4 – CSS Basics

CSS (Cascading Style Sheets) makes HTML beautiful. On Day 4, I learned selectors, colors, fonts, and spacing. Styling gave me creative control over the plain HTML.

My FabLab experience: I enjoyed experimenting with background colors and typography. Watching text instantly transform after adding a CSS rule felt magical. I also realized the importance of separating content from style. 

h1 { color: purple; }
p { font-size: 16px; }
#main { background: black; }

Day 5 – Layouts with Flexbox & Grid

Day 5 was about positioning. Flexbox helps align items in rows and columns, while Grid handles complex 2D layouts. These systems make responsive design possible.

During practice, I used Flexbox to center elements both vertically and horizontally—something that used to be very hard with older CSS. I also built a 3-column Grid layout, which showed me how powerful CSS has become. 

.container {
  display: flex;
  justify-content: space-between;
}
.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

Day 6 – Animations & Transitions

On Day 6, I explored how to add life to web pages using CSS transitions and animations. Transitions make hover effects smooth, while animations can move elements over time.

My experience: I animated a button to glow when hovered. It made the page feel modern and interactive. I learned that subtle animations can guide user focus without being distracting. 

button {
  transition: background 0.3s ease;
}
button:hover {
  background: #7b5cff;
  color: white;
}

Day 7 – JavaScript Basics

JavaScript is the brain of a website. I learned about variables, functions, and how to interact with the DOM (Document Object Model). This allows dynamic updates to content without reloading the page.

At FabLab, I wrote a script that changed text on a button click. Debugging my first error messages was frustrating, but when it finally worked, I felt empowered. This session showed me how logic and creativity meet in coding. 

// Variables
let name = "Tanushri";

// Function
function greet(user) {
  return "Hello, " + user + "!";
}

console.log(greet(name));

// DOM example
document.getElementById("title").innerText = "Welcome to FabLab Web Dev!";
