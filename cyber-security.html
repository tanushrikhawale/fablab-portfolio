Cyber Security & Networking

This 7-day combined module gave me deep insights into computer networking and cyber security practices. I learned how data flows across devices, how networks are built, and how to secure them against modern threats. The journey mixed theory, labs with Cisco Packet Tracer, and practical security with Kali Linux.

Day 1 – Networking Basics

Introduced to different network types (LAN, WAN, MAN) and devices such as hubs, switches, and routers. Learned the difference between IP addresses and MAC addresses, and how data is broken into packets for transmission.

Activity: Drew a basic home network diagram and traced how a smartphone connects to a laptop via a WiFi router.

Day 2 – OSI Model & TCP/IP

Explored the 7-layer OSI Model and the 4-layer TCP/IP stack. Understood how protocols like HTTP, FTP, TCP, UDP, and IP fit into the model. Learned the concept of encapsulation and de-encapsulation during communication. 

ping 192.168.1.2

Activity: Connected two PCs in Packet Tracer, assigned IP addresses, and verified connectivity with ping.

Day 3 – IP Addressing & Subnetting

Learned about IPv4 address classes, private vs public IPs, and subnet masks. Practiced subnetting by dividing 192.168.1.0/24 into smaller subnets for efficient use. 

PC1: 192.168.1.10  Subnet: 255.255.255.0  Gateway: 192.168.1.1
PC2: 192.168.1.20  Subnet: 255.255.255.0  Gateway: 192.168.1.1

Activity: Configured manual IP addresses and successfully tested communication.

Day 4 – Routing & Switching Labs

Learned the role of switches (Layer 2) and routers (Layer 3). Configured static routing to connect two LANs in Packet Tracer. 

Router> enable
Router# configure terminal
Router(config)# ip route 192.168.2.0 255.255.255.0 10.0.0.2

Activity: Built two LANs, connected them with routers, and enabled cross-LAN communication.

Day 5 – Cyber Security Fundamentals

Learned about the CIA Triad (Confidentiality, Integrity, Availability) and different attack types like Phishing, DDoS, and SQL Injection. Explored security defenses such as firewalls, IDS/IPS, and antivirus software.

Activity: Analyzed a phishing email example and listed signs to detect suspicious links.

Day 6 – Hands-on Security

Used Kali Linux tools to perform network scanning and traffic analysis. Practiced ethical hacking basics with Nmap and Wireshark. 

# Scan open ports
nmap 192.168.1.1

# Capture live traffic
wireshark

# Display IP configuration
ifconfig

Activity: Captured HTTP traffic in Wireshark and observed why unencrypted websites are unsafe.

Day 7 – Secure Networking Practices

Final day focused on best practices like strong password policies, enabling HTTPS, using SSL/TLS certificates, and VPNs for secure communication. Discussed real-world cases like WannaCry ransomware and how organizations defend against such attacks.

Activity: Configured WPA2 security on routers in Packet Tracer and enabled firewalls for protection.

✅ Learning Outcomes

  • Mastered OSI model, IP addressing, and subnetting.
  • Configured routers & switches in Packet Tracer.
  • Understood common cyber threats and countermeasures.
  • Performed basic penetration testing with Kali Linux tools.
  • Designed and secured small-scale networks.
Mentor: Gorav Sharma guided me through both Networking and Cyber Security, ensuring I gained a balance of theoretical knowledge and practical skills.
